# Instruções da Atividade

Este repositório faz parte da atividade prática da disciplina de **Documentação Técnica em DevOps**.

## Objetivo do grupo
Documentar a infraestrutura da CloudHealth de modo que equipes técnicas consigam operar, evoluir e auditar o ambiente com segurança.

## Documentação que deve ser produzida
- Arquivo final: `docs/infrastructure.md`
- Referência de estrutura: `docs/infrastructure-template.md`

## Regras da simulação
- Esta atividade é uma simulação acadêmica.
- Não é necessário provisionar recursos reais nem desenvolver código.
- O foco está na documentação técnica de infraestrutura.

## Forma de trabalho
- O trabalho deve ser realizado no fork do grupo.
- Decisões e premissas devem ser explicitadas quando faltarem dados.
