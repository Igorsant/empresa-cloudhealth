# empresa-cloudhealth

## Setor
HealthTech

## Contexto do problema
A CloudHealth opera serviços críticos de dados clínicos e agenda médica em nuvem. O crescimento da plataforma ocorreu sem documentação consolidada de infraestrutura, dificultando gestão de capacidade, segurança, continuidade e conformidade.

## Objetivo do desafio
Produzir documentação técnica de infraestrutura com visão de ambientes, componentes, segurança, operação e riscos.

## Instruções resumidas para os grupos
1. Faça um fork deste repositório para a conta do grupo.
2. Trabalhe exclusivamente no fork do grupo.
3. Preencha a documentação na pasta `docs/`, usando `docs/infrastructure-template.md` como referência e consolidando o resultado em `docs/infrastructure.md`.
4. Entregue o link do fork do grupo conforme orientação da disciplina.
